Configurar acesso ao repositório de dados

Crie em GitHub → Settings → Developer settings → Fine-grained tokens, restrito a este repositório, com permissões "Contents: Read" e "Actions: Read and write". O token fica salvo apenas no localStorage do seu navegador — nunca é enviado a nenhum servidor além da API do GitHub.
